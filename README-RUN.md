# Run Your Site - Super Simple!

Just run this ONE command:

```bash
./run.sh
```

That's it! It will:
1. Install everything
2. Build the site
3. Start the server

Then open: **http://localhost:8000**

---

**Or if you prefer to do it manually:**

```bash
npm install --legacy-peer-deps && npm run build && python3 serve.py
```
